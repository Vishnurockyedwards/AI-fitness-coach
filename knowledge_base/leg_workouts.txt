Leg workouts: muscles, examples, and structure

Main leg muscles
- Quadriceps: knee extension (e.g., squats, leg press, lunges, leg extensions).
- Hamstrings: hip hinge + knee flexion (e.g., Romanian deadlifts, leg curls).
- Glutes: hip extension/abduction (e.g., hip thrusts, squats, lunges).
- Calves: plantarflexion (e.g., standing/seated calf raises).

Example beginner leg session (2–3x/week)
- Goblet squat: 3 sets x 8–12 reps
- Romanian deadlift (dumbbells): 3 x 8–12
- Reverse lunge: 2–3 x 8–10 per leg
- Calf raises: 3 x 12–20
- Optional: plank or dead bug for core stability

Technique cues
- Squat: brace core, knees track over toes, keep heels down, control depth you can maintain.
- Hinge (RDL): push hips back, keep spine neutral, feel stretch in hamstrings, keep weights close.
